Data and API Catalog Schema

The beginnings of Socrata's proposed modifications to the proposed DCAT and Data Catalog Schema and Protocol.

Our proposal is heavily based on the work previously done by the W3C on the DCAT standard and CKAN's proposed modifications to that standard in the Data Catalog Schema and Protocol. It also incorporates some of Data.gov's work on API catalogs.

Files of note:

  • specification.mkd - The detailed spec (work currently in progress)
  • notes.mkd - Various notes from our learnings
  • examples/ - Examples of catalog entries and listings in different formats
